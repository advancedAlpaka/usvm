# JVM TI Agent для ConcolicHelper

Этот агент устанавливает брейкпоинт на вход в метод `breakpoint()` класса `org.usvm.instrumentation.ConcolicHelper`.

## Структура проекта

- `concolic_agent.cpp` - Основной файл агента JVM TI
- `Makefile` - Скрипт сборки
- `TestConcolic.java` - Тестовая программа
- `build_and_run.sh` - Скрипт для сборки и запуска

## Сборка

Для сборки агента выполните:

```bash
make
```

Это создаст файл `libconcolic_agent.so`.

## Использование

Для запуска Java-программы с этим агентом используйте:

```bash
java -agentpath:./libconcolic_agent.so YourMainClass
```

## Как это работает

1. Агент отслеживает загрузку класса `org/usvm/instrumentation/ConcolicHelper`
2. При загрузке класса агент находит метод `breakpoint()`
3. Агент устанавливает брейкпоинт на вход в этот метод
4. При срабатывании брейкпоинта выводится сообщение в консоль

## Тестирование

Для тестирования агента выполните:

```bash
./build_and_run.sh
```

Это:
1. Соберет агент
2. Скомпилирует тестовую программу
3. Запустит тестовую программу с агентом

## Настройка

Агент не требует дополнительных параметров конфигурации.
